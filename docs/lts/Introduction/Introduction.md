# 长期技术支持

![alt text](image.png)

如上图，此项赞助可以解锁一些常见游戏的Unity解密功能，方便手里没有解密脚本的用户使用。

不过得提前说明几个细节：

- 所有的解密功能，都是直接复刻其它人开源或者分享给我的解密脚本而得到，优点是转换成Rust代码了，执行速度很快。
- LTS赞助的本质是辛苦费和最低的技术维护费，如果你懒得去自己搞解密脚本，可以把这部分任务交给AssetDaemon来做，可以节省你一部分的时间。
- 由于AssetDaemon本身不优先维护解密功能，此功能不能保证所有的解密功能都对应最新版本的游戏文件，它只是帮你省去了辛苦搜罗的时间。
- 你可以把某游戏的解密脚本发给我，交由我添加到AssetDaemon中方便你后续使用，也方便其它人使用。
- 作者本身不会，也不具备破解游戏得到解密算法的能力，所有解密功能均来自互联网信息收集，所以说这个本质上是辛苦费，如果全网都找不到解密算法的话作者也没办法。

所以，此功能在我看来并不是必须的功能，因为大部分使用AssetDaemon的用户都有自己的解密渠道，如果没有或者要把解密嵌入进AssetDaemon，就可以赞助LTS来节省你这方面的时间。

此外，加入LTS支持计划后，你的反馈建议会被第一优先级考虑并加入到工具中，也会有一个私密性质的社群，更方便交流分享。

最后：LTS的意义本质上在于给用户提供更好的体验，无法做到任何游戏任何功能都能完美实现的完美解决方案，如果有兴趣支持本工具开发，可以来赞助。
